Detailed Description

[Inherited from QDialog] Class that defines the window to add a Couple to a Relation.

Retrieve all the Relations and Notes in order for the user to create a new Couple. Checks if the Couple is already present (with regard to the orientation of the Relation selected) before adding it in. The Relation 'Référence' isn't proposed since its lives throught direct références in texts.
